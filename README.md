# ⚖️ Lucas Evaristo - Advocacia e Consultoria

Este é um site institucional funcional desenvolvido para o meu projeto acadêmico, focado em apresentar o trabalho jurídico do escritório de advocacia parceiro.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica.
* **CSS3:** Layouts com Grid e Flexbox.
* **JavaScript:** Validação de formulários e interatividade.

## 🛠️ Funcionalidades
- [x] Cabeçalho fixo
- [x] Hero section com imagem de fundo e overlay.
- [x] Grade de áreas de atuação (Criminal, Trabalhista, etc).
- [x] Seção de perfil do advogado responsável.
- [x] Formulário de contato funcional (validação via JS).
- [x] Rodapé com horários de funcionamento.
